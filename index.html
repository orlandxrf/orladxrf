Experience

Ph.D Dissertation Project

Named Entity Recognition, Relation Extraction, Logic Rules, and Knowledge Graph

The project consists of knowledge discovery from unstructured documents (online newspapers' articles in the Spanish language). Machine learning and deep learning was used to build a model to identify and recognize named entities. The entities are used for the Relation Extraction task for every two entities inside one sentence. A logic rules group was defined to infer new facts. Finally, the facts integrate to make a knowledge graph.
